Installation de l'API

Suivez ces étapes pour installer et lancer le serveur API MuSE localement.


1. Pré-requis

Assurez-vous d'avoir Python 3.8+ et PostgreSQL installés.

2. Récupérer le projet Git

Clonez le projet depuis GitHub :

git clone https://github.com/Pikoow/SAE_C_25-26.git

Sinon, téléchargez le projet depuis GitHub.

3. Installation des dépendances

Dans votre terminal, installez les librairies nécessaires :

pip install -r requirements.txt

4. Configuration

Créez un fichier .env à la racine du projet avec vos accès BDD :

POSTGRES_DBNAME=votre nom de db  
POSTGRES_USER=votre nom de user  
POSTGRES_PASSWORD=votre password  
POSTGRES_PORT=votre port

5. Mise en place de la base de données

Récuperez les CSV nettoyés via ce lien Google Drive et placez-les dans le dossier Python/script_peuplement du projet.

Exécutez le script de peuplement à la racine du projet pour créer les tables et insérer les données :

python setup_db.py

6. Lancer le serveur API

Une fois la base de données configurée, lancez le serveur API avec :

python API/scripts/main.py

7. Accéder au site

Dans un terminal différent, lancez le serveur node avec :

node API/web/node-auth/server.js

Ouvrez votre navigateur et accédez à http://localhost:3000 pour voir le site en action.
